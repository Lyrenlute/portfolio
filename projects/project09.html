Project Description

Adara is a passion project focused on world-building with the goal of making a fluid Action Adventure / TPS RPG (or at least a playable demo). This is a collaborative project with all ip being used by permission of the owner.

Tasks

My role in this game project was that of technical lead, taking responsibility for leading the entirety of the game design and implementation. My first steps included the thorough write-up of a comprehensive GDD. Central aspects to describe were features such as gameplay, mechanics, story and world design.

Implementation was paused at an early stage of proof of concept levels in the Unity game engine. Focus on the narrative side and backstory was deemed to require additional focus before proceeding. Development has reached the point at which it is easy to continue with robust documentation and a large base of gathered assets.
